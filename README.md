# Scoop Bucket

Personal Scoop bucket for Windows package management.

## Usage

```powershell
scoop bucket add licht1stein https://github.com/licht1stein/scoop-bucket
scoop install sanskrit-upaya
```

## Available Apps

| App                                                             | Description                                                                                    |
| --------------------------------------------------------------- | ---------------------------------------------------------------------------------------------- |
| [sanskrit-upaya](https://github.com/licht1stein/sanskrit-upaya) | Cross-platform Sanskrit dictionary with fast full-text search across 36 digitized dictionaries |
